# Contributing to Aqua Sophia

Thank you for your interest in contributing to Aqua Sophia!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/your-username/aqua-sophia.git`
3. **Create** a branch: `git checkout -b feature/your-feature`
4. **Make** your changes
5. **Test** your changes
6. **Submit** a Pull Request

- Use meaningful names
